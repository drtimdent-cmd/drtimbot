# Бот записи для Doctor Tim Dental

Рабочий прототип Telegram-бота для записи пациентов. Хранит пациентов и записи в локальной базе SQLite (файл `clinic.db` создаётся автоматически).

## Как запустить

1. **Получи токен бота**
   - Открой Telegram, найди @BotFather
   - Отправь `/newbot`, придумай имя (например "Doctor Tim Dental Bot")
   - Скопируй токен, который он выдаст

2. **Узнай свой Telegram ID**
   - Найди @userinfobot, напиши ему что угодно — он покажет твой ID

3. **Настрой окружение**
   ```
   cp .env.example .env
   ```
   Открой `.env` и впиши:
   - `BOT_TOKEN=` — токен из шага 1
   - `ADMIN_CHAT_ID=` — твой ID из шага 2

4. **Установи зависимости и запусти**
   ```
   npm install
   node index.js
   ```

5. Напиши своему боту `/start` в Telegram — должна заработать запись.

## Что уже умеет

- Выбор услуги из списка (консультация, чистка, протезирование, пломба)
- Выбор свободного времени из расписания
- Подтверждение записи пациенту
- Уведомление администратору (тебе) о каждой новой записи
- Команда `/appointments` — список всех записей (доступна только тебе)
